# New York Taxi Analysis using Machine Learning

## Overview
This project explores the New York City taxi trip dataset to understand ride patterns and predict outcomes like fare amount and trip duration. The goal is to turn raw trip data into meaningful insights using simple and effective machine learning models.

## What This Project Does
* Cleans and prepares real-world taxi trip data
* Performs exploratory data analysis (EDA)
* Builds ML models such as Linear Regression and Random Forest
* Evaluates model performance using standard metrics (MAE, MSE, R²)

## Tools Used
Python, Pandas, NumPy, Matplotlib, Scikit-learn

## Why This Project Matters
Transportation data is complex and large-scale. This project demonstrates how machine learning can be applied to solve practical problems like fare prediction and demand analysis in a clear and structured way.
